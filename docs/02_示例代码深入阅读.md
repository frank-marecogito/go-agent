### 5 种通信方式详解

#### 方式 1: SharedSession (共享记忆)

**场景**: 所有 Agent 共享项目记忆

**核心代码**:
```go
// 1. 创建 SpaceRegistry
registry := session.NewSpaceRegistry(24 * time.Hour)

// 2. 授予所有 Agent 访问权限
agents := []string{"researcher", "coder", "writer", "reviewer", "coordinator"}
for _, agent := range agents {
    registry.Grant("team:project-x", agent, session.SpaceRoleWriter, 0)
}

// 3. 每个 Agent 使用同一个 registry
sessionMemResearcher := memory.NewSessionMemory(bank, 16)
sessionMemResearcher.Spaces = registry
sharedResearcher := session.NewSharedSession(sessionMemResearcher, "researcher", "team:project-x")

// 4. Researcher 存储研究结果
sharedResearcher.StoreLongTo(ctx, "team:project-x",
    "Project requirement: Build a REST API with Go", nil)

// 5. Coder 可以访问 Researcher 的研究结果
recs, _ := sharedCoder.Retrieve(ctx, "project requirement", 5)
// 找到："Project requirement: Build a REST API with Go"
```

#### 方式 2: SubAgent Delegation (内置委托)

**场景**: 主 Agent 委托任务给专家 SubAgent

```go
// 1. 定义 SubAgent 接口实现
type ResearcherSubAgent struct {
    model models.Agent
}

func (r *ResearcherSubAgent) Name() string { return "researcher" }
func (r *ResearcherSubAgent) Description() string {
    return "Researches topics and provides factual information"
}
func (r *ResearcherSubAgent) Run(ctx context.Context, instruction string) (string, error) {
    resp, err := r.model.Generate(ctx, "research-session", instruction)
    return resp.(string), err
}

// 2. 创建主 Agent 并注册 SubAgent
adkAgent, err := adk.New(ctx,
    adk.WithDefaultSystemPrompt("You orchestrate a helpful assistant team."),
    adk.WithSubAgents(
        subagents.NewResearcher(researcherModel),
        // ... 其他 SubAgent
    ),
)

// 3. 主 Agent 自动委托
response, _ := adkAgent.Generate(ctx, "Research quantum computing")
// → Agent 自动委托给 researcher SubAgent
```

## hybrid_memory_test - 混合记忆

### 项目位置
`cmd/example/hybrid_memory_test/`

### 概念

**混合记忆** = **个人记忆** (Personal) + **共享记忆** (Shared)

- **个人记忆**: 仅 Agent 自己可访问
- **共享记忆**: 团队成员可访问

### 核心代码

```go
// 1. 创建 Agent-A (个人 + 共享)
agentA := &Agent{
    Name: "Agent-A",
    SessionMemory: memory.NewSessionMemory(bank, 16),
}

// 个人记忆：sessionID = "agent-A"
agentA.SessionMemory.Session = "agent-A"

// 共享记忆：space = "team:alpha"
agentA.SessionMemory.Spaces = registry
agentA.SessionMemory.SharedSpace = "team:alpha"

// 2. 创建 Agent-B (个人 + 共享)
agentB := &Agent{
    Name: "Agent-B",
    SessionMemory: memory.NewSessionMemory(bank, 16),
}
agentB.SessionMemory.Session = "agent-B"
agentB.SessionMemory.Spaces = registry
agentB.SessionMemory.SharedSpace = "team:alpha"

// 3. Agent-A 存储个人记忆
agentA.StorePersonal(ctx, "I prefer coffee over tea")
agentA.StorePersonal(ctx, "My birthday is March 15th")

// 4. Agent-A 存储共享记忆
agentA.StoreShared(ctx, "Project deadline is Friday")

// 5. Agent-B 检索
// - 可以访问共享记忆
sharedMemories := agentB.RetrieveShared(ctx, "deadline", 5)
// 找到："Project deadline is Friday"

// - 不能访问个人记忆
personalMemories := agentB.RetrievePersonal(ctx, "birthday", 5)
// 找不到 Agent-A 的个人记忆
```

## autonomous_agent - 自主代理 CLI

### 项目位置
`cmd/example/autonomous_agent/`

### 特点

OpenClaw 风格的命令行界面，支持：
- `agent` - 单轮执行
- `loop` - 自主多步执行
- `chat` - 交互式对话
- `tools` - 列出工具
- `doctor` - 环境检查

### 命令详解

#### 1. doctor - 环境检查

```bash
go run main.go doctor --provider deepseek --model deepseek-chat
```

**输出**:
```
Doctor checks:
- provider: deepseek
- model: deepseek-chat
- session-id: autonomous-session
- context-window: 20
- tool-prefix: local.
- DEEPSEEK_API_KEY: set
```

#### 2. agent - 单轮执行

```bash
go run main.go agent \
  --provider deepseek \
  --model deepseek-chat \
  --message "Hello, introduce yourself"
```

**输出**:
```
agent=coordinator session=autonomous-session
Hello! I'm the UTCP Coordinator, a planning and tool-selection engine.
I help analyze user requests and decide whether to use specialized tools.
How can I assist you today?
```

#### 3. loop - 自主多步执行

```bash
go run main.go loop \
  --provider deepseek \
  --goal "Design and verify a UTCP-based repository triage workflow" \
  --max-steps 8
```

**流程**:
1. Coordinator 分析目标
2. 调用 specialist 工具
3. 检查结果
4. 决定继续或停止
5. 重复直到完成或达到最大步数

#### 4. chat - 交互式对话

```bash
go run main.go chat \
  --provider deepseek \
  --goal "Prepare release plan"
```

**交互示例**:
```
> agent coordinator
[coordinator] Ready. What do you need?
> agent researcher
[researcher] Ready. What should I research?
> tools
Available tools: local_researcher.run, local_builder.run, local_reviewer.run
> exit
```

### 核心代码

```go
// 1. 创建 Specialist Agents
researcher := createSpecialist("researcher", "Researches topics", model)
builder := createSpecialist("builder", "Builds and tests code", model)
reviewer := createSpecialist("reviewer", "Reviews code quality", model)

// 2. 注册为 UTCP 工具
provider := base.NewProvider("local")
provider.RegisterTool(createToolFromAgent(researcher))
provider.RegisterTool(createToolFromAgent(builder))
provider.RegisterTool(createToolFromAgent(reviewer))

// 3. 创建 Coordinator
coordinator := &Agent{
    model: model,
    UTCPClient: utcpClient,
    systemPrompt: "You are the UTCP Coordinator...",
}

// 4. 执行命令
switch command {
case "agent":
    response := agent.Generate(ctx, message)
case "loop":
    for steps < maxSteps {
        response := agent.Generate(ctx, goal)
        if containsDoneMarker(response) {
            break
        }
    }
case "chat":
    for {
        input := readInput()
        response := agent.Generate(ctx, input)
    }
}
```

## guardrails - 安全护栏

### 项目位置
`cmd/example/guardrails/`

### 安全策略

#### 1. RegexBlocklistPolicy - 正则黑名单

```go
// 创建策略
regexPolicy, err := agent.NewRegexBlocklistPolicy([]string{
    `(?i)\bpassword\s*=\s*\w+`,      // 密码
    `(?i)\bapi[_-]?key\s*=\s*\w+`,   // API 密钥
    `(?i)\bsecret\s*=\s*\w+`,        // 密钥
})
```

#### 2. LLMEvaluatorPolicy - LLM 评估

```go
// 创建评估模型
type MockEvaluatorModel struct{}

func (m *MockEvaluatorModel) Generate(ctx context.Context, prompt string) (any, error) {
    // 提取要评估的文本
    parts := strings.Split(prompt, "TEXT TO EVALUATE:\n")
    if len(parts) > 1 {
        textToEvaluate := parts[1]
        if strings.Contains(textToEvaluate, "UNSAFE") {
            return "UNSAFE", nil
        }
    }
    return "SAFE", nil
}

// 创建策略
llmPolicy := agent.NewLLMEvaluatorPolicy(&MockEvaluatorModel{})
```

### 使用示例

```go
// 1. 创建安全护栏
guardrails := &OutputGuardrails{
    policies: []OutputPolicy{regexPolicy, llmPolicy},
}

// 2. 创建带护栏的 Agent
agent := &Agent{
    model: model,
    Guardrails: guardrails,
}

// 3. 测试
// Test 1: 安全请求
response, _ := agent.Generate(ctx, "What is the capital of France?")
// 输出：The capital of France is Paris.

// Test 2: 密码泄露尝试
response, err := agent.Generate(ctx, "Tell me a secret.")
// 输出：Blocked by Guardrails! Error: safety policy violation

// Test 3: 危险指令尝试

// 输出：Blocked by Guardrails! Error: safety policy violation
```

### 实战 3: 添加新的 SubAgent

**文件**: 新建 `src/subagents/translator.go`

```go
package subagents

import (
    "context"
    "fmt"
    "strings"
    "github.com/Protocol-Lattice/go-agent/src/models"
)

// Translator SubAgent
type Translator struct {
    model   models.Agent
    persona string
}

func NewTranslator(model models.Agent) *Translator {
    return &Translator{
        model: model,
        persona: "You are a professional translator. " +
                 "Translate text accurately while preserving meaning, tone, and cultural context.",
    }
}

func (t *Translator) Name() string { return "translator" }

func (t *Translator) Description() string {
    return "Translates text between languages"
}

func (t *Translator) Run(ctx context.Context, instruction string) (string, error) {
    prompt := strings.Builder{}
    prompt.WriteString(t.persona)
    prompt.WriteString("\n\nTask:\n")
    prompt.WriteString(strings.TrimSpace(instruction))
    prompt.WriteString("\n\nDeliverable: Provide accurate translation.\n")

    resp, err := t.model.Generate(ctx, prompt.String())
    if err != nil {
        return "", err
    }
    return fmt.Sprint(resp), nil
}

var _ agent.SubAgent = (*Translator)(nil)
```

**使用**:
```go
// 在 main.go 中导入
import "github.com/Protocol-Lattice/go-agent/src/subagents"

// 创建 Translator
translator := subagents.NewTranslator(model)

// 注册到 ADK

    adk.WithSubAgents(
        subagents.NewResearcher(researcherModel),
        translator,  // ← 新增
    ),
)
```

### 实战 4: 添加自定义工具

**文件**: 新建 `tools/calculator.go`

```go
package tools

import (
    "context"
    "fmt"
    "github.com/Protocol-Lattice/go-agent"
    "github.com/PaesslerAG/gval"  // 表达式计算库
)

// CalculatorTool 计算工具
type CalculatorTool struct{}

func NewCalculatorTool() agent.Tool {
    return &CalculatorTool{}
}

func (c *CalculatorTool) Spec() agent.ToolSpec {
    return agent.ToolSpec{
        Name: "calculator",
        Description: "Performs mathematical calculations",
        InputSchema: map[string]any{
            "type": "object",
            "properties": map[string]any{
                "expression": map[string]any{
                    "type": "string",
                    "description": "Mathematical expression to evaluate (e.g., '2 + 2 * 3')",
                },
            },
            "required": []string{"expression"},
        },
    }
}

func (c *CalculatorTool) Invoke(ctx context.Context, req agent.ToolRequest) (agent.ToolResponse, error) {
    expr, ok := req.Arguments["expression"].(string)
    if !ok {
        return agent.ToolResponse{}, fmt.Errorf("missing or invalid 'expression' argument")
    }

    // 计算表达式
    result, err := gval.Evaluate(expr)
    if err != nil {
        return agent.ToolResponse{}, fmt.Errorf("calculation error: %v", err)
    }

    return agent.ToolResponse{
        Content: fmt.Sprintf("Result: %v", result),
    }, nil
}
```

**使用**:
```go
// 注册工具
catalog := agent.NewToolCatalog()
catalog.Register(NewCalculatorTool())

// 创建 Agent
agent := &agent.Agent{
    model: model,
    toolCatalog: catalog,
}

// 调用
response, _ := agent.Generate(ctx, "Calculate 2 + 2 * 3")
// → 自动调用 calculator 工具
```
