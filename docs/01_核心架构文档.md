## 项目概览

### 基本信息

- **名称**: Lattice (go-agent)
- **语言**: Go 1.25+
- **用途**: 构建 AI 代理的开发框架
- **核心特性**:
  - 🧩 模块化架构 - 通过声明式配置组合代理
  - 🤖 多代理支持 - 专家代理通过共享目录协调

## Agent 核心实现

### 文件位置
`agent.go` (1819 行)

### 核心结构体

```go
type Agent struct {
    // LLM 模型
    model        models.Agent
    
    // 记忆系统
    memory       *memory.SessionMemory
    systemPrompt string
    contextLimit int

    // 工具目录
    toolCatalog       ToolCatalog
    subAgentDirectory SubAgentDirectory
    UTCPClient        utcp.UtcpClientInterface

    // 共享记忆
    Shared    *memory.SharedSession
    
    // CodeMode 工作流
    CodeMode  *codemode.CodeModeUTCP
    CodeChain *chain.UtcpChainClient

    // 安全护栏
    Guardrails       *OutputGuardrails
    AllowUnsafeTools bool

    // 互斥锁
    mu sync.Mutex
}
```

### 核心方法

#### 1. Generate - 单次生成

```go
func (a *Agent) Generate(ctx context.Context, prompt string) (any, error)
```

**流程**:
1. 加载会话记忆
2. 构建提示词（系统提示 + 用户输入 + 记忆）
3. 调用 LLM 模型
4. 解析工具调用（如果有）
5. 执行工具
6. 返回结果

#### 2. GenerateWithFiles - 带文件上下文

```go
func (a *Agent) GenerateWithFiles(ctx context.Context, prompt string, files []models.File) (any, error)
```

**特点**:
- 支持多模态输入（文本 + 文件）
- 自动处理文件编码
- 内联文本文件，引用非文本文件

#### 3. GenerateStream - 流式生成

```go
func (a *Agent) GenerateStream(ctx context.Context, prompt string) (<-chan models.StreamChunk, error)
```

**特点**:
- 支持流式输出
- 实时显示生成过程
- 可中途取消

### 工具调用流程

```go
// 1. 检查是否需要调用工具
if containsToolCall(response) {
    // 2. 解析工具调用
    toolCalls := parseToolCalls(response)
    
    // 3. 执行工具
    for _, call := range toolCalls {
        result := a.InvokeTool(ctx, call)
        
        // 4. 将结果添加到对话
        conversation = append(conversation, result)
    }
    
    // 5. 再次调用模型生成最终回复
    finalResponse := a.Generate(ctx, conversation)
}
```

### SubAgent 委托

```go
// 1. 检查是否委托给 SubAgent
if subAgent, ok := a.subAgentDirectory.Get(toolName); ok {
    // 2. 调用 SubAgent
    result := subAgent.Run(ctx, instruction)
    
    // 3. 返回结果
    return result
}
```

### 创建流程

```go
// 1. 创建 ADK
adk, err := adk.New(ctx,
    // 2. 配置 SubAgent
    adk.WithSubAgents(subagents.NewResearcher(researcherModel)),
    
    // 3. 配置模块
    adk.WithModules(
        // 模型模块
        adkmodules.NewModelModule("gemini-model", func(_ context.Context) (models.Agent, error) {
            return models.NewGeminiLLM(ctx, "gemini-2.5-pro", "Swarm orchestration:")
        }),
        
        // 记忆模块
        adkmodules.InQdrantMemory(100000, *qdrantURL, *qdrantCollection, memory.AutoEmbedder(), &memOpts),
    ),
)

// 4. Bootstrap 引导（执行所有模块）
if err := kit.Bootstrap(ctx); err != nil {
    log.Fatal(err)
}
```

### 记忆引擎

#### 重要性评分

```go
type ScoreWeights struct {
    RecencyWeight    float64  // 新近度权重 (默认 0.3)
    ImportanceWeight float64  // 重要性权重 (默认 0.5)
    RelevanceWeight  float64  // 相关性权重 (默认 0.2)
}
```

**评分公式**:
```
final_score = recency_score * recency_weight +
              importance_score * importance_weight +
              relevance_score * relevance_weight
```

#### MMR 检索 (Maximal Marginal Relevance)

```go
// 平衡相关性和多样性
func mmr(query string, candidates []*MemoryRecord, lambda float64, k int) []*MemoryRecord
```

**参数**:
- `lambda`: 相关性权重 (0-1, 默认 0.5)
- `k`: 返回数量

type Tool interface {

### ToolSpec

```go
type ToolSpec struct {
    Name        string         // 工具名称
    Description string         // 工具描述
    InputSchema map[string]any // JSON Schema 输入定义
}
```

### 示例：计算器工具

```go
type CalculatorTool struct{}

func (c *CalculatorTool) Spec() ToolSpec {
    return ToolSpec{
        Name: "calculator",
        Description: "Performs mathematical calculations",
        InputSchema: map[string]any{
            "type": "object",
            "properties": map[string]any{
                "expression": map[string]any{
                    "type": "string",
                    "description": "Mathematical expression to evaluate",
                },
            },
            "required": []string{"expression"},
        },
    }
}

func (c *CalculatorTool) Invoke(ctx context.Context, req ToolRequest) (ToolResponse, error) {
    expr := req.Arguments["expression"].(string)
    result := evaluateExpression(expr)
    return ToolResponse{Content: result}, nil
}
```

## 安全护栏

### 文件位置

### OutputGuardrails

```go
type OutputGuardrails struct {
    policies []OutputPolicy
}

type OutputPolicy interface {
    CheckOutput(ctx context.Context, output string) error
}
```

### 预置策略

#### 1. RegexBlocklistPolicy - 正则黑名单

```go
// 创建策略
regexPolicy, err := agent.NewRegexBlocklistPolicy([]string{
    `(?i)\bpassword\s*=\s*\w+`,      // 密码
    `(?i)\bapi[_-]?key\s*=\s*\w+`,   // API 密钥
    `(?i)\bsecret\s*=\s*\w+`,        // 密钥
})
```

#### 2. LLMEvaluatorPolicy - LLM 评估

```go
// 创建评估模型
evaluatorModel := &MockEvaluatorModel{}

// 创建策略
llmPolicy := agent.NewLLMEvaluatorPolicy(evaluatorModel)

// 添加到护栏
guardrails.AddPolicy(llmPolicy)
```

### 使用示例

```go
// 创建带护栏的 Agent
agent := &Agent{
    model: model,
    Guardrails: &OutputGuardrails{
        policies: []OutputPolicy{regexPolicy, llmPolicy},
    },
}

// 调用时自动检查
response, err := agent.Generate(ctx, prompt)
if err != nil {
    // 可能是安全策略拦截
    if strings.Contains(err.Error(), "safety policy violation") {
        // 处理拦截
    }
}
```
